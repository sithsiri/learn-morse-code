Learn Morse Code!

This project aims to be a simple, easy-to-use Morse code practice game. I haven't been able to find something similar to this and have been wanting to make it for a long time. Its goals are to be simple and intuitive, and more useful than just staring at the charts. Check out the code here, or just press start to learn Morse code! For a bigger challenge, disable learning mode.
